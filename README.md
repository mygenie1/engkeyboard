# 한영 학습 키보드

일상 타이핑 중 자연스럽게 영어 단어에 노출되고 학습하는 안드로이드 키보드입니다.
번역기나 치환 도구가 아니며, **사용자가 입력한 텍스트는 절대 바꾸지 않습니다.**

- 언어/기술: Kotlin, Android InputMethodService 기반 네이티브 키보드
- 최소 안드로이드: 8.0 (minSdk 26)
- 네트워크: **사용 안 함** (인터넷 권한 자체가 없음)

## 개발 진행 상황

| 단계 | 내용 | 상태 |
|------|------|------|
| 1 | 프로젝트 뼈대 + 자동 빌드(GitHub Actions) | ✅ 완료 |
| 2 | 두벌식 자판 UI + 한글 오토마타 | ✅ 완료 |
| 3 | 한글 오토마타 단위 테스트 137개 (도깨비불 포함) | ✅ 완료 |
| 4 | 학습 사전(SQLite) + 추천 바 + 학습 카드 | ✅ 완료 |
| 4.5 | 영어 QWERTY 자판 + 숫자·기호판 + 한/영 전환 | ✅ 완료 |
| 5 | 설정 화면(소리/진동/키 높이) + 앱 아이콘 | ⬜ 예정 |

## APK(설치 파일)는 어떻게 받나요?

이 저장소에 코드가 올라올 때마다 GitHub가 자동으로 설치 파일을 만들어 둡니다.

1. GitHub 저장소 페이지 → 상단 **Actions** 탭 클릭
2. 가장 최근 실행(초록색 체크) 클릭
3. 아래 **Artifacts** 항목의 `app-debug-apk` 를 다운로드 → 압축 해제하면 `app-debug.apk`

## 폰에 설치하고 키보드 켜는 법

1. `app-debug.apk` 를 폰으로 옮겨 설치 (알 수 없는 앱 설치 허용 필요)
2. 폰 **설정 → 일반 → 언어 및 입력 → 화면 키보드 → 키보드 관리**
   (기종마다 메뉴 이름이 조금 다릅니다)
3. **한영 학습 키보드** 를 켜기(ON)
4. 아무 앱에서 입력창을 누른 뒤, 키보드 전환 버튼으로 **한영 학습 키보드** 선택
