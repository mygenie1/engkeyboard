# Handoff: 틈새영어학습 키보드 — 비주얼 테마(1a) + 앱 아이콘(2b) 적용

## Overview
안드로이드 네이티브(Kotlin) 키보드 앱 "틈새영어학습 키보드"의 확정 디자인 스펙입니다.
- **키보드 테마: 1a "톤온톤 청회 (Quiet Slate)"** — 라이트/다크 2벌
- **앱 아이콘: 2b "틈새 블록"** — 두 키 블록이 비껴 놓여 사이에 틈이 생기는 형태

이 문서 하나로 구현이 가능하도록 모든 수치를 dp / sp / HEX / 굵기로 명시했습니다.

## About the Design Files
동봉된 `키보드 비주얼 3안.dc.html`은 **HTML로 만든 디자인 레퍼런스(목업)**입니다. 그대로 앱에 넣는 코드가 아니며, 이 스펙을 **기존 Kotlin 코드베이스의 방식(XML 레이아웃/Compose, 기존 키보드 뷰 구조)** 안에서 재현하는 것이 과제입니다. 캔버스에서 1a 열과 2b 카드가 확정안입니다 (1b, 1c, 2a는 탈락안 — 참고용).

## Fidelity
**High-fidelity.** 색·크기·굵기·간격은 아래 수치를 그대로 적용하세요.

## 2. 키보드 본체

### 레이아웃 수치 (dp)
- 키 높이: **보통 46 / 낮음 40 / 높음 52** (기존 높이 3단계 설정과 매핑) · 가로 모드 30
- 키 간격: 가로 5 · 세로 7
- 키 모서리 반경: 7
- 키보드 좌우 패딩: 4
- 키 그림자(라이트): y1 blur0, 검정 16% — 은은한 1dp 하단 엣지. 다크: y1 blur0, 검정 40%
- 배치·줄 수는 **현행 유지** (숫자줄 포함 5줄, 기능 키 위치 변경 금지)

### 키 글자 (Roboto / Noto Sans KR 시스템 폰트)
- 한글·영문 키: **18sp Regular(400)**, 색 keyText
- 숫자줄: **16sp Regular(400)**
- 기능 키 텍스트(?123, 한/영, 가/ABC, 1/2, 2/2): **12sp Medium(500)**, 색 fnText
- 글리프 키(⇧ ⌫ ⏎): 19sp
- Caps Lock 상태는 ⇪로 표시 (현행 로직 유지)

### 눌림 피드백
- 눌린 키 배경을 즉시 **pressed** 색으로 교체, 손 떼면 즉시 복귀
- **애니메이션·트랜지션 금지** (입력 지연 0이 원칙)

## 3. 추천 바 + 추천 칩
- 바 높이: **40dp 고정** (칩이 없어도 유지, 배경은 bg와 동일 — 별도 구분선 없음)
- 칩: 높이 **30dp**, 반경 **15dp**(완전 라운드), 좌우 패딩 **12dp**, 칩 간격 **7dp**, 좌측 시작 여백 8dp
- 칩 글자: **14sp Medium(500)** chipText / 배경 chipBg / 테두리 1dp chipBorder
- 영어 모드 칩의 보조 뜻(예: "사과 · 명사"): **12sp Regular(400)**, 색 sub, 단어와 6dp 간격
- 칩 4개 이상: 좌우 스와이프, 마지막 칩이 우측에서 절반 잘리게 배치해 "더 있음" 암시

## 4. 학습 카드 (오버레이)

### 공통
- 자판 영역(추천 바 제외) 위에 오버레이. **카드 높이 ≤ 자판 영역 높이** (키보드 전체 높이 불변 — 기존 자동 테스트 유지)
- 배경 cardBg, 반경 **10dp**, 패딩 상하 **12dp** 좌우 **16dp**
- 그림자: y4 blur16, 검정 20%
- 우상단 ✕: 14sp, 색 sub, 터치 영역 최소 44×44dp
- 좌상단 컨텍스트 라벨(예: "한→영 · 방금 입력: 사과"): 12sp Regular, 색 sub
- 내용 초과 시 카드 내부 스크롤 (카드 크기 불변)
- 카드 하단 광고 자리: 기본 높이 0 유지. 펼쳐질 경우 50dp 배너가 카드 하단에 붙고 본문이 그만큼 스크롤 영역이 됨

### 1군 풀 카드 (위에서부터)
1. 표제어: **24sp Bold(700)** keyText + 발음기호 **14sp Regular** accent (baseline 정렬, 간격 10dp)
2. 품사·뜻: **15sp Regular** keyText, 상단 여백 6dp (예: "명사 · 사과")
3. 구분선: 1dp divider, 상하 여백 10dp
4. 예문: **14sp Regular** keyText, 줄간 1.5 / 번역: **13sp Regular** sub, 상단 여백 3dp

### 2군 간이 카드
풀 카드에서 발음기호·예문 줄을 **뷰 자체를 제거**(GONE)해 빈 줄 없이 축소.

### 다의어 카드
1. 표제어(한글): 24sp Bold(700)
2. 항목 리스트 (배 → pear / stomach / ship):
   - 영어 단어 **16sp Medium(500)** keyText + 발음기호 **13sp Regular** accent + 우측 끝 분야 태그 **11sp Regular** sub (예: 과일 / 신체 / 선박)
   - 항목 상하 패딩 9dp, 항목 사이 1dp divider

## 5. 기호판
- 색·수치는 본체와 동일. 1/2 ↔ 2/2 전환 키와 가/ABC 키는 기능 키 스타일(fnBg/fnText, 12sp Medium)
- 기호 글자는 18sp Regular — 배치는 현행 유지

## 6. 설정 화면
- 배경 `#F7F9FB`, 상단 헤더는 흰 배경 + 하단 1dp `#E7ECF1`
- 헤더: 앱 아이콘 26dp + "틈새영어 키보드" 16sp SemiBold(600) `#1E2833`
- 섹션 라벨(입력 / 학습 / 정보): **11sp Bold(700)**, 색 accent `#46698C`, 자간 +0.06em, 상단 여백 14dp
- 항목: 제목 **15sp Regular** `#1E2833` / 부제 **12sp Regular** `#6C7A8A`, 상하 패딩 11dp, 좌우 16dp
- 스위치: 36×20dp 트랙(반경 10dp), ON `#46698C` / OFF `#C6CDD6`, 흰 노브 16dp
- 항목 순서: [입력] 키 입력음 · 진동 · 키보드 높이(낮음/보통/높음) / [학습] 추천 바 표시 / [정보] 사전 출처·라이선스(국립국어원 한국어기초사전 · CC BY-SA 표기 필수) · 앱 정보(버전 · "인터넷 권한 없음" 명시)

## 7. 앱 아이콘 — 2b "틈새 블록"
Android adaptive icon으로 제작 (foreground / background 분리, 벡터 드로어블 권장).

- **배경 레이어**: 단색 `#FFFFFF`
- **전경 레이어** (108dp 캔버스, 안전 영역 66dp 기준 — 아래 수치는 96dp 타일 환산값이므로 비율로 적용):
  - 블록 1 "한": 38×38, 반경 10, 배경 `#46698C`, 글자 흰색 Bold(700) 17 — **좌상단** (타일 기준 left 14, top 14)
  - 블록 2 "A": 38×38, 반경 10, 배경 `#202B36`, 글자 흰색 Bold(700) 17 — **우하단** (right 14, bottom 14)
  - 두 블록 사이 대각선 틈 ≈ 6 — 이 여백이 "틈새"의 핵심. 블록이 겹치거나 틈이 넓어지면 안 됨
- **테마 아이콘(monochrome 레이어, Android 13+)**: 두 블록 실루엣만 단색으로
- 표시명: "틈새영어 키보드" (잘림 방지 축약명)

## 8. Interactions & Behavior (변경 없음 — 회귀 방지 체크)
- 사용자 입력 텍스트는 절대 수정하지 않음 (카드·추천은 표시 전용)
- 추천 유지/교체/소멸 규칙, 활용형 인식, 백스페이스 자모 삭제 + 길게 눌러 가속 삭제: 현행 로직 그대로
- 한/영/기호 전환 시 조합 중 한글 확정 처리 유지
- 카드 닫힘: ✕ / 카드 밖 탭 / 타이핑 재개
- 모드 전환·카드 표시·추천 바 유무 어느 경우에도 **키보드 전체 높이 불변**
- 네트워크 리소스 사용 금지 (시스템 폰트만, 원격 이미지 없음)

## 9. 검증 체크리스트
- [ ] 라이트/다크 각각에서 한글·영어·기호판 렌더 확인
- [ ] 높이 3단계 × 세로/가로 조합 (특히 **낮음 + 가로**에서 카드가 자판 높이를 넘지 않는지)
- [ ] 칩 0개일 때 추천 바 40dp 유지
- [ ] 2군 간이 카드에서 빈 줄 미노출
- [ ] 눌림 피드백에 애니메이션이 없는지 (색 교체만)
- [ ] 런처·설정·키보드 선택 목록에서 아이콘 및 "틈새영어 키보드" 표시 확인
